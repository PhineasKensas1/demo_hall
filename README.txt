Mode d'emploi de l'Application de Dessin
Dessiner des Formes:

Cliquez sur l'onglet "Dessiner".
Choisissez une forme dans le menu "Dessiner" (Rectangle, Ellipse, Cercle, Carré, Losange, Hexagone).
Pour les formes autres que le cercle, entrez la largeur et la hauteur.
Choisissez une couleur en utilisant la boîte de dialogue de couleur.
Cliquez sur le canevas pour placer la forme.
Sélectionner et Éditer des Formes:

Cliquez sur une forme sur le canevas pour la sélectionner.
Double-cliquez sur une forme pour éditer ses propriétés (dimensions et couleur).
Appuyez sur la touche "Supprimer" pour supprimer la forme sélectionnée.
Déplacer des Formes:

Cliquez sur une forme et faites-la glisser pour la déplacer sur le canevas.
Double-cliquez sur une forme pour activer le mode édition, puis déplacez-la.
Enregistrement et Chargement:

Cliquez sur l'onglet "Fichier".
Sélectionnez "Enregistrer" pour sauvegarder le dessin actuel (choisissez un nom et un emplacement).
Sélectionnez "Charger" pour charger un dessin précédemment enregistré.
Remarques Importantes:

Pour les cercles, la largeur spécifiée représente le diamètre.
Le périmètre et la surface des formes sont automatiquement calculés et affichés.
L'onglet "Éditer" permet de modifier les propriétés d'une forme sélectionnée.
